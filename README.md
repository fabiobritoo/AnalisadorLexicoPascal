# Analisador Léxico

Desenvolvimento de um Analisador Léxico para a linguagem Pascal

## Tipos de dados:

1. Palavras Reservadas
2. Números Inteiros
3. Números Reais
4. Variáveis
5. Símbolos

## Instalação

Abra cmd na pasta

- python -m venv env
- cd env/Scripts
- activate.bat
- cd..
- cd..
- pip install -r requirements.txt
- python analisador_pascal.py
